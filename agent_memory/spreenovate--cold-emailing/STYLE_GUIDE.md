# Style Guide — Spreenovate Cold Emails

## Tonalität
- Direkt, aber respektvoll
- Peer-to-Peer, nicht Verkäufer-zu-Kunde
- Neugierig, nicht pushy
- Deutsch (Du/Sie je nach Branche — bei Corporates: Sie, bei Startups: Du möglich)

## Aufbau einer Email

### Betreffzeile
- Max. 8 Wörter
- Provokant oder neugierig machend
- Bezug zur Person/Firma
- KEINE Clickbait-Taktiken

### Body (max. 120 Wörter)
1. **Hook** (1-2 Sätze): Konkreter Bezug zur Person/Firma. Zeigt, dass recherchiert wurde.
2. **Brücke** (1-2 Sätze): Verbindung zum Thema KI/Automatisierung. Was könnte relevant sein?
3. **Wert** (1 Satz): Was Spreenovate konkret machen kann — kein generischer Pitch.
4. **CTA** (1 Satz): Offene Frage, kein Druck. "Hätten Sie Lust auf einen kurzen Austausch?"

### Signatur
```
Beste Grüße
Alexander Kamphorst
```

## Checkliste
- [ ] Betreffzeile < 8 Wörter?
- [ ] Body < 120 Wörter?
- [ ] Konkreter Bezug zur Person/Firma im Hook?
- [ ] Kein generischer KI-Pitch?
- [ ] Frage als CTA, nicht Aufforderung?
- [ ] Keine Buzzwords (Synergie, revolutionär, disruptiv)?

## Beispiel

**Betreff:** Wenn KI Sozialkompetenz misst — Konkurrenz oder Ergänzung?

**Body:**
Hallo Herr Haddick,

ich habe gesehen, dass DNLA seit Jahren Sozialkompetenz messbar macht. Spannend.

Mit dem Aufkommen von KI-basierten Assessments stellt sich die Frage: Ergänzung oder Konkurrenz?

Ich helfe Unternehmen wie Ihrem, KI strategisch einzusetzen — nicht als Ersatz, sondern als Verstärker.

Hätten Sie Lust auf einen kurzen Austausch?

Beste Grüße
Alexander Kamphorst
