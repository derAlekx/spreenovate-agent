# Agent Memory — Spreenovate Cold Emailing

## Wer ist Spreenovate?
- KI-Beratung aus Berlin, gegründet von Alexander Kamphorst
- Fokus: KI-Workflows und Automatisierung für KMU im DACH-Raum
- Nicht: KI-Entwicklung/ML-Training, sondern pragmatische Anwendung bestehender KI-Tools

## Was funktioniert gut
- Personalisierte Hooks basierend auf konkreter Recherche (Firmennews, Produkte, LinkedIn)
- Kurze Emails (max. 120 Wörter Body)
- Fragen am Ende statt harte CTAs
- Deutsch als Sprache (DACH-Markt)

## Was NICHT funktioniert
- Generische "KI ist toll"-Pitches ohne konkreten Bezug
- Zu lange Emails
- Zu viele Buzzwords
- Englisch bei deutschen Kontakten

## Zielgruppe
- Geschäftsführer, CTOs, Heads of Product bei KMU (10-500 MA)
- Branchen: SaaS, HR-Tech, E-Commerce, Agenturen, Beratungen
- Pain Points: Interne Prozesse manuell, Content-Produktion skalieren, KI-Features einbauen
